# PHOTON-Chat-Room-LAN (BETA)

## English

**PHOTON-Chat-Room-LAN (BETA)** is a terminal-based chat room written in C++ using Boost.Asio.

### Server
- Asynchronously accepts connections.  
- Reads messages from clients.  
- Broadcasts messages to all connected clients.  

### Client
- Reads messages from the server.  
- Sends messages to the server.  
- Uses multithreading to handle reading and writing simultaneously.  

This project is a BETA version and works on local networks (LAN).

## Русский

**PHOTON-Chat-Room-LAN (BETA)** — это чат-комната, работающая в терминале, написанная на C++ с использованием Boost.Asio.

### Сервер
- Асинхронно принимает подключения.  
- Читает сообщения от клиентов.  
- Рассылает сообщения всем подключённым клиентам.  

### Клиент
- Получает сообщения от сервера.  
- Отправляет сообщения на сервер.  
- Использует многопоточность для одновременного чтения и отправки сообщений.  

Проект находится в версии BETA и предназначен для работы в локальной сети (LAN).
